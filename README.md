# 비나우 공용 법인차량 예약 대시보드

GitHub에 올리고 Vercel에 연결하면 바로 확인 가능한 Next.js 대시보드입니다.

## 실행 방법

```bash
npm install
npm run dev
```

브라우저에서 아래 주소로 접속합니다.

```bash
http://localhost:3000
```

## Vercel 배포

1. GitHub 저장소에 이 폴더의 파일을 업로드합니다.
2. Vercel에서 해당 GitHub 저장소를 Import 합니다.
3. Framework Preset은 Next.js로 선택합니다.
4. Deploy를 누릅니다.

## 현재 버전의 저장 방식

현재 버전은 Slack 연동 전 화면/기능 확인용으로 `localStorage`에 예약 정보를 저장합니다.

즉, 같은 브라우저에서는 새로고침해도 예약이 유지되지만, 여러 직원이 동시에 공유하는 실제 운영용으로 쓰려면 Supabase, Vercel Postgres, Neon 같은 DB를 붙여야 합니다.

다음 단계에서 `/api/reservations`와 DB 테이블을 추가하면 현재 화면 구조를 거의 그대로 유지한 채 실사용 버전으로 확장할 수 있습니다.

## 구현된 기능

- Pretendard 폰트 적용
- 전체 / 그랜져 / 레이 탭
- 월별 캘린더
- 예약 완료 날짜 색상 표시
- 예약 완료 날짜의 같은 차량 중복 클릭 방지
- 예약 폼: 이름, 직책, 소속, 차량, 예약일자/시간, 반납일자/시간
- 안전운행 확인 체크박스
- 평일 예약 상시 허용
- 주말 예약은 해당 예약월의 전월 마지막 월요일 오전 10시부터 허용
- 예약 완료 후 캘린더에 예약자명/소속 표시
